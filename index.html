Mesonet Station Status

Live reporting status for the Montana Mesonet network

What is this?

The live reporting state of every station in the Montana Mesonet — HydroMet and AgriMet sub-networks — operated by the Montana Climate Office.

Visualization modes

  • Status — single threshold: dot is "fresh" if the station posted within the last hour, "stale" otherwise.
  • Time since — five bins of minutes since the latest record, ceiling at >24 h.

Color comes from Crameri's roma scientific colour map — perceptually uniform, color-vision-deficiency safe, and ordered green = good → red = bad.

Controls

  • Mode toggle — Switch between Status and Time-since.
  • Network chips — Toggle HydroMet / AgriMet visibility; counts show the total stations per sub-network.
  • Labels — Show or hide station IDs next to each dot. Labels dodge each other to avoid overlap.
  • Search — Type a station name or ID and pick a suggestion to fly there and open its popup.
  • Zoom to extent — Snap the map back to fit Montana.
  • Theme — Toggle light/dark.
  • Legend — Click the chevron in the lower-left to collapse or expand.

Map interactions

  • Hover a dot — Tooltip with station name, ID, last-record timestamp, and relative time (in your local timezone).
  • Click a dot — Popup with details and links to the station dashboard + latest data.
  • Co-located sites — When multiple stations share a location, the HydroMet station is shown with a count badge. Hover (desktop) or tap (mobile) to fan the others out as a "spider"; a second click on a foot opens that station's popup.
  • Esc — Closes any open popup or spider.

Sharable URLs

Every piece of UI state — map center, zoom, mode, networks, labels, legend, theme, and the currently-open station — is reflected in the URL. Copy it from the address bar to share or bookmark; loading the URL restores the same view. To deep-link to a station, use ?station=aceabsar.

Data sources

  • Stations: /api/stations/
  • Latest records: /api/latest/

Data auto-refresh every 5 minutes; dot colors update every 30 seconds against the local clock.
